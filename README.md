User Generated Orbit (UGO) is a little vibe coding project to visualize a user's movement in Google Maps. It is based on an initial idea from 2007 to visualize movement in Google Earth.

Have a try at: https://tobedetermined.github.io/ugo

See the old blog from 2007 at: http://www.usergeneratedorbit.com/
